Deep Learning





Have you ever thought about how Netflix knows which shows, series, movies you might be interested in, or how Amazon knows what products you may like in advance, or how Apple’s Siri , Google Voice Assistant or Microsoft’s Cortana understands and assist you? It is not magic or some sorcery, rather those are complex algorithms that analyze patterns and human behavior which is further processed to serve with intelligent results.

Artificial Intelligence makes a computer system or a robot think like a human. These are the set of algorithms that train the computer system to learn themselves, and give them the intellect to make human-like decisions.

Deep Learning is a subfield of machine learning which further comes under artificial intelligence. Machine Learning is the set of algorithms that mimic the structure and functional unit of human brain. Data is the most important parameter for artificial intelligence. We may have succeeded in developing algorithms that gives highly accurate results but deep learning may fail if quality of data is not checked. For example: To classify a dog’s image, there should be images of every dog breed and images of other 4-legged animals so that the system learns to differentiate dogs from other animals.

Deep Learning was introduced by Rina Dechter in 1986. The first general, operating algorithmic rule for supervised learning, deep learning, feedforward, multilayer perceptrons was revealed by Alexey Ivakhnenko, and Lapa in 1965 but it was not practiced and followed due to the scarcity of large amount of data and hardware to process the data. With the evolution of information technology, the large amount of data has been generated in the last two decades and fast computing devices like GPU (graphical processing unit) have been developed.

In recent years, Deep learning has developed tremendously. It is used in the field of computer vision, natural language processing, speech recognition, medicine, drug discovery, games, and social network analysis where results were remarkable and supercilious when compared to human intelligence. Due to its achievements in various fields, it has drawn attention from various leaders and experts in business industry who are further making advancements in it. Evidently, it is the most buzzed topic in the industry.

Deep learning differs from traditional machine learning where we need to provide the features as input to the algorithms. Deep learning can automatically learn features from data such as images, video, text, sound without introducing hand-coded rules or human domain knowledge. This process of deep learning is called “feature extraction”.



Deep Learning uses multi-layered artificial neural networks where it consists of an input layer, many hidden layers, and the output layer. The “deep” word signifies that input data propagates through many numbers of layers and transforms into an abstract and composite representation. Each layer contains nodes that represent the part of the information in that level.

The Input layer contains nodes equal to the number of dimensions in the input data. There are different number of nodes in each layer. The number of nodes in the output layer equals the number of classes in the classification problem. However, in regression problems, there is a single node in the output layer. The nodes in hidden layers and output layers are parameterized for abstract information i.e. features of input change during learning. Activation functions in each layer assist in converting the input signals to output signals. Output signals of layer are further used as input for another layer. The final output layer gives the final output.



Learning is merely updating node weights(parameters) to a new one and the updation is defined by a loss function(e.g SVM or Softmax or Cross entropy Loss). Loss function is defined as the difference between the real value and the estimated value of the output. Minimization of loss function is required for accuracy. Loss value is propagated back to previous layer neurons where the new value of weights are computed using current loss values. This process is called as Back propagation.

Machine Learning

Input Data — — — → Feature Extraction — — — → Algorithm — — — → Output

Example: (Vehicle image) — — — → (Features Defined) — — — → (Classification) — — — → (Car or Non Car)

Deep Learning

[ Input Data ] — — — → [ Feature Extraction + Algorithm ] — — — → [ Output ]

Example: (Vehicle image) — — — → (Features Extraction + Classification) — — — → (Car or Non Car)
